# SPPC - Soluzione di Piccoli Problemi in C

### PROBLEMA:
	Gioco del TRIS o TIC TAC TOE:
	Implementare il classico gioco del tris, nel quale lo scopo è allineare
	consecutivamente nel campo di gioco 3 propri segni in orizzontale,
	verticale o diagonale, impedendo all'avversario di fare altrettanto.

### REQUISITI:
	1. Possibilità di giocare utente contro utente, utente contro computer e
	   computer contro computer.
	2. Gestione degli input non validi dell'utente.
	3. Mosse del computer con elaborazione di 1 turno successivo alla mossa.
