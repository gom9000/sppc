# Soluzione di Piccoli Problemi in C

### PROBLEMA:
	Gioco dell'impiccato:
	Implementare il classico gioco dell'impiccato, nel quale il giocatore deve
	indovinare una parola misteriosa, scegliendo via via le lettere che la
	compongono.
	La tradizione vuole che ad ogni lettera errata proposta dal giocatore venga
	disegnato un "pezzetto" di forca alla quale, esaurite le mosse a disposizione,
	penzolerà il povero giocatore sconfitto.


### REQUISITI:
	1. La parola da indovinare viene selezionata, dal programma ed in modo
	   casuale, da un elenco di parole contenute in un file di testo allegato.
	2. Ad ogni mossa vengono visualizzate tutte le lettere inserite dal giocatore,
	   sia errate che esatte.
	3. Ad ogni errore del giocatore viene visualizzato un pezzetto di forca.
	4. Il programma termina, con un appropriato messaggio, quando il giocatore
	   indovina la parola o quando esaurisce le mosse a disposizione.
